## Python Calculator (2024)

A desktop calculator application built using Python and Tkinter.

## Features

* Addition, subtraction, multiplication, and division
* Also includes functions such as squaring and square root
* User-friendly graphical interface
* Error handling for invalid expressions
* Lightweight and easy to use

## Technologies Used

* Python
* Tkinter

## Executable Version

A standalone Windows executable is available in the Releases section.

## Learning Outcomes

This project was built in 2024 as a school project while learning:

* Python programming
* GUI development with Tkinter
* Event-driven programming
* Application packaging with PyInstaller
